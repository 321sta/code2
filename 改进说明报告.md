# 冬小麦智能灌溉预测模型改进说明报告

## 一、改进背景

原始模型为 **IBA-ELM**（改进蝙蝠算法优化极限学习机），通过改进蝙蝠算法（IBA）对 ELM 的输入权重与隐层偏置进行联合优化，在冬小麦日蒸散量（ETc）预测任务上取得了较好的效果。

然而，标准 ELM 的输出权重 β 通过 Moore-Penrose 伪逆直接求解：

$$\beta = H^+ y = (H^T H)^{-1} H^T y$$

这种方式在训练样本有限、特征共线性较强时容易出现过拟合或数值不稳定，泛化能力受限。

---

## 二、改进内容

### 2.1 引入 L2 正则化（岭回归）

将 ELM 输出权重的求解方式从伪逆改为**岭回归（Ridge Regression）**：

$$\beta = (H^T H + \lambda I)^{-1} H^T y$$

其中 λ 为正则化系数，对输出权重的 L2 范数加以约束，有效抑制过拟合、提升泛化能力。

### 2.2 将 λ 纳入 IBA 搜索空间联合优化

传统做法需要人工网格搜索确定 λ，耗时且难以保证最优。本次改进将 `log10(λ)` 作为搜索向量的第 `dim+1` 维，与 ELM 输入权重、偏置**联合优化**：

| 搜索向量结构 | 维度 |
|---|---|
| 输入权重 W_in（拉平） | n_in × n_hidden |
| 隐层偏置 b | n_hidden |
| **log₁₀(λ)（新增）** | **1** |

λ 的搜索范围设为 `log10(λ) ∈ [-4, 2]`，即 **λ ∈ [10⁻⁴, 100]**，由 IBA 自适应确定，无需人工调参。

### 2.3 改进前后对比

| 项目 | IBA-ELM（原始） | IBA-ELM（改进后） |
|---|---|---|
| ELM 输出权重求解 | Moore-Penrose 伪逆 | L2 正则化岭回归 |
| 正则化系数 λ | 无 | IBA 联合优化自动确定 |
| 搜索向量维度 | n_in×n_hidden + n_hidden | n_in×n_hidden + n_hidden + **1** |
| 过拟合风险 | 较高 | 显著降低 |
| λ 调参方式 | 不适用 | 自动 |

## 三、实验结果

### 3.1 与基线模型的对比（第五章）

在全量数据（2023—2025 年两期冬小麦生育期数据）上的训练集指标对比：

| 模型 | R² | RMSE (标准化) | MAE (标准化) |
|---|---|---|---|
| **IBA-ELM（改进后）** |  0.9033 | 0.3110 | 0.2443 |
| IBA-ELM（原始） | 0.9002 | 0.3160 | 0.2524 |
| BA-ELM | 0.8820 | 0.3435 | 0.2683 |
| ELM | 0.8583 | 0.3764 | 0.2994 |
| MLP | 0.8438 | 0.3952 | 0.2976 |

## 四、可视化系统（Streamlit 应用）

在模型改进的基础上，开发了基于 **Streamlit** 的冬小麦智能灌溉需水预测系统，主要功能如下：

### 4.1 单次预测
- 输入6项气象参数（日均气温、相对湿度、大气压、风速、太阳辐射、作物系数 Kc）
- 输出当日 ETc 预测值（mm/day）及灌溉等级建议

### 4.2 灌溉建议分级

| ETc 范围 | 灌溉建议 |
|---|---|
| < 1.5 mm/day | 无需灌溉 |
| 1.5 ~ 3.0 mm/day | 轻度灌溉 |
| 3.0 ~ 5.0 mm/day | 中度灌溉 |
| ≥ 5.0 mm/day | 大量灌溉 |

### 4.3 批量预测
- 支持上传 CSV 文件进行批量逐日预测
- 可视化预测曲线，支持与实测值对比
- 结果可一键下载

### 4.4 模型信息展示
- 模型参数、性能指标一览
- IBA 优化收敛曲线展示
